# react+springboot《视频网站》

## 目录结构(详细的说明在各个项目文件下的README.md)
1. javabishe是包含了前台后台的java代码
    * admin是后台项目
    * core是核心项目
    * front-desk是前台项目
2. my-admin是后台界面
3. my-frontDesk是前台界面

## 实现功能
* 前台
    * 登陆
    * 注册
    * 分类显示视频
    * 视频最新推荐
    * 视频播放
* 后台
    * 登陆，验证
    * 用户查看，冻结
    * 视频查看
    * 前台头部图片查看，添加
    * 管理员
        * 后台人员查看，删除，新增
        * 角色查看，删除，新增

## BUG

## 运行
1.  admin项目  
    访问http://localhost:8089/
2.  front-desk项目  
    访问http://localhost:8080/
3.  my-admin项目  
    下载依赖  
    `npm install`  
    运行  
    `npm start`
    访问http://localhost:8000/
4.  my-frontDesk项目  
    下载依赖  
    `npm install`    
    运行  
    `npm start`    
    访问http://localhost:8079/
